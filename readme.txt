== Description ==

This plugin will regenerate all missing image sizes for an image when it's chosen as a featured image for any post. Very useful when you change your template. No settings whatsoever, just activate and forget.

== Installation ==

1. Download and Upload the plugin to the `/wp-content/plugins/` directory (or just use `wp-admin` installer.)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done!
